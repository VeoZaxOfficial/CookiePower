# 🍪 CookiePower

A fun plugin for PocketMine-MP (API 2.0.0) that gives players special abilities when eating cookies.

## ✨ Features
- Heals player when eating a cookie
- Gives Speed effect
- Cool flame particle effects

## 📦 Installation
1. Download the plugin
2. Put it in your `/plugins/` folder
3. Restart your server

⬇ Download Link: [Click Here To Download](https://www.mediafire.com/file/pwclprql5v94lh7/CookiePower_v0.14.3_0.15.10.phar/file)

## 🎮 How it works
- Eat a cookie 🍪
- Get:
  - ❤️ +6 health
  - ⚡ Speed boost
  - 🔥 Flame particles around you

## ⚙️ Requirements
- PocketMine-MP (0.14.x- 0.15.10)
